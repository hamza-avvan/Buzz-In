🛠️ Installation

Requirements

  • Node.js 14+
  • npm 6+

Steps

# 1. Clone repository
git clone https://github.com/your-repo/quiz-show.git
cd quiz-show

# 2. Install dependencies
npm install

# 3. Add sound files (optional)
mkdir -p public/sounds
cp path/to/sounds/* public/sounds/

# 4. Start server
node server.js
